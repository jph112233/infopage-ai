# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1YiSKrlbENM_cA7bWDkPzTqvRYZvROLK3

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `OPENROUTER_API_KEY` in [.env.local](.env.local) to your OpenRouter API key
   - Get your API key from [OpenRouter.ai](https://openrouter.ai/settings/keys)
3. Run the app:
   `npm run dev`
